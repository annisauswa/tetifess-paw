## Deskripsi
Media sosial untuk pengiriman pesan anonim dari dan kepada mahasiswa DTETI FT UGM

## Persiapan Server
```
cd ./server
npm run dev
```

menyalakan server selamanya untuk mencegah crash
```
forever start server.js
forever list
forever stop server.js
```
